Predicting House Prices

In this project, I developed a predictive model to estimate house prices using a real estate dataset. Employed both multiple linear regression and polynomial regression techniques to analyse key features such as square footage, property grade, waterfront views, and location (encoded by zip code). Performed feature engineering to encode categorical variables and normalise numerical features. Evaluated model performance using metrics like R-squared and RMSE, comparing the accuracy of linear and polynomial models. Presented findings with data visualisations to highlight model insights and key predictors of house prices.
